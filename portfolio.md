<h2>Online Reader | Orc Reader</h2>
<a href="http://dzul.datorklubs.lv/orcreader/demo2.html"><img class="alignnone size-full wp-image-1512" title="Suara-Merdeka---Orc-Reader-1" src="http://subject9.files.wordpress.com/2011/08/suara-merdeka-orc-reader-1.jpg" alt="" width="590" height="458" /></a>

This is a JavaScript application I created using mostly jQuery and CSS. It was intended for use as an e-reader for online newspapers and magazines. The first example shows one potential usage as an online newspaper reader. This example includes page linking as well as URL linking. Pages are dynamically loaded as the user progresses, but this can be altered to force all the pages to be preloaded on page load.

<a href="http://dzul.datorklubs.lv/orcreader/demo1.html"><img class="alignnone size-full wp-image-1513" title="Battle-Chasers-#1---Orc-Reader-0" src="http://subject9.files.wordpress.com/2011/08/battle-chasers-1-orc-reader-0.jpg" alt="" width="590" height="455" /></a>

The second example shows how the Orc Reader can be used as an e-comic reader. This can also be extended to be used for e-magazines, e-graphic novels and so on. Both examples use a small PHP script that creates thumbnails according to the preset size, which can be altered when setting up the app.

Link: <a title="Orc Reader Demo #1" href="http://dzul.datorklubs.lv/orcreader/demo1.html" target="_blank">Demo#1</a> <a title="Orc Reader Demo #2" href="http://dzul.datorklubs.lv/orcreader/demo2.html" target="_blank">Demo#2</a>

Skills involved: JavaScript, jQuery, CSS, PHP
